ansible批量安装宝塔脚本

2024-04-04
阅读时长: 1 分钟

摘要GPT
摘要小助理暂时失联跑路啦……😜

版本介绍：7.7.0版本 包括一些优化

脚本

https://gitee.com/dayu777/open_shell/raw/main/tb.yaml

运行

ansible-playbook tb.yaml

手动破解

先登录面板，然后破解，去除登陆需要绑定账号：

rm -f /www/server/panel/data/bind.pl

手动解锁宝塔所有付费插件为永不过期

sed -i 's/"endtime": -1/"endtime": 999999999999/g' /www/server/panel/data/plugin.json

给plugin.json文件上锁防止自动修复为免费版

chattr +i /www/server/panel/data/plugin.json

去后门

sudo echo "" > /www/server/panel/script/site_task.py
sudo chattr +i /www/server/panel/script/site_task.py
sudo rm -rf /www/server/panel/logs/request/*
sudo chattr +i -R /www/server/panel/logs/request

卸载宝塔

wget http://download.bt.cn/install/bt-uninstall.sh

执行

sh bt-uninstall.sh
